Trainingssessie

Opdracht

Deze opdracht komt uit het OPO Professionele ontwikkeling 2. De bedoeling is om een praktische IT gerelateerde trainingsessie te organiseren op maat van de cursisten.


Uitwerking

Ik en 2 medestudenten hebben een trainingsessie van 2 uur gegeven in het Sancta Maria intituut te Aarschot aan de 6e jaars van de studieriching informatica. Wij hebben besloten om in deze praktische training het Bootstrap framework met focus op het grid systeem te behandelen tijdens het 1ste lesuur. Waarom ? Omdat bootstrap wereldwijd het meeste gebruikte web framework is en het grid systeem belangrijk is om mooi gestructureerde responsive websites te bouwen. De cursisten hadden al ervaring met html en css dus 1 lesuur was voldoende om het grid systeem met Bootrstrap onder de knie te krijgen. Het 2e lesuur hebben wij versiebeheer met Git en GitHub ge√Øntroduceerd. We hebben de studenten leren samenwerken aan een webproject met 2 of 3 personen waarbij versiebeheer via git en github verliep. Dankzij deze training hebben de studenten zeker en vast een streepje voor in het hoger onderwijs waarbij ze in een IT gerelateerde studierichting zeker met versiebeheer in aanraking gaan komen. Deze trainingssessie gaf ik samen met 2 medestudenten. Ieder had zijn eigen deel van de cursus die hij aan de cursisten ging uitleggen. tijdens de oefeningen waren wij alle 3 aanspreekbaar voor eventuele vragen en om hulp te bieden aan aan de cursisten.


Besluit

Dankzij deze opdracht heb ik modulair cursusmateriaal leren uitwerken als ook het opleiden van de cursisten zelf aan de hand van dit zelf geschreven cursusmateriaal. Dankzij de informatieve vragen en hier en daar wat vragen over de oefeningen van de cursisten heb ik ook leren inspelen op onverwachte situaties.
