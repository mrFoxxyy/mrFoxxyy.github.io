Website Console Knowledge

Opdracht

Bij het OPO Webdesign met html5 kregen wij de opdracht om een fictief webburea op te richting en met jouw team een informatieve website rondom game consoles uit te werken. Deze website moest gemaakt zijn met het Bootstrap framework waarvan we enkele variabelen naar eigen wensen moesten aanpassen. Ook SASS en Jquery als ook CSS3 animaties horen aan bod te komen in deze website. Versiebeheer gebeurd natuurlijk via Git en GitHub.



Uitwerking

Met mijn fictief webbureau met als CEO ikzelf begonnen we met volle moed aan het uitwerken van deze website. Verschillende componenten die het Bootstrap framework ons bied hebben we mooi kunnen toepassen in onze site. We zorgden Voor gepaste CSS3 animaties waar het kon en werkte met Jquery enkele functies uit die zorgden voor iets meer interactie met de gebruiker. Jquery, Bootstrap, SASS, CSS3 en versiebeheer met Git en GitHub zijn de technologiën die en vaardigheden die in dit project sterk aan bod kwamen en die ik dus ook goed beheers. Omdat ik CEO was van ons fictief webbureau was het aansturen van het team en zorgen voor een goede taakverdeling een belangrijke taak. Daarnaast hield ik ook het versiebeheer via GitHub goed in de gaten zodat ik kon monitoren wanneer iemand een toevoeging deed aan de website en of deadlines gerespecteerd werden. Onderaan de pagina kan u screenshots zien van het eindresultaat van onze website.


Besluit

Dankzij dit project heb ik vooral veel bijgeleerd over het coderen van hedendaagse websites met eerder vernoemde technologiën. De leerstof over deze technologiën beheers ik zeker voldoende om praktisch hiermee aan de slag te gaan. Verder heb ik als CEO van het webbureau ook een klein team van in totaal 3 personen leren aansturen. Omdat het team bestond uit mensen die wel eens durven uitstellen was dit op sommige momenten zeker een uitdaging.
